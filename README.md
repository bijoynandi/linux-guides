# Fedora KDE Setup

## Prerequisites

1. Copy `.env.example` to `.env`:
```bash
   cp .env.example .env
```

2. Edit `.env` and set your passwords:
```bash
   nano .env
```

3. Source environment variables:
```bash
   source .env
```

4. Run setup scripts!

## Important

- **Never commit `.env` to git!**
- `.env` is in `.gitignore` for safety
- Use `.env.example` as template
